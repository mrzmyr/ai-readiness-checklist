# GENERATED_ARTIFACTS

Status: `IF_APPLICABLE`

## Requirement

Generated files are listed with source-of-truth inputs, regeneration command, and "do not hand edit" rule.

## Acceptance Criteria

- Generated paths are listed.
- Source inputs are listed.
- Regeneration command is exact.
- Hand-editing is forbidden.

## Automatable Checks

- If generated directories exist, docs mention regeneration command.
- `AGENTS.md` or `docs/generated.md` mentions "do not edit" or equivalent.

## Example

```md
- `src/generated/` is generated by `pnpm codegen`; do not edit by hand.
- Source inputs live in `schema/*.graphql`.
- After changing schema files, run `pnpm codegen && pnpm check`.
```
